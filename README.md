# 🤣 Chistes-Nicolas

Repositorio de chistes con gestión completa (CRUD) y consulta aleatoria por categoría.

## Stack

- **Frontend:** React + Vite
- **Backend:** Node.js
- **Base de datos:** PostgreSQL
- **Infraestructura:** Docker + docker-compose
